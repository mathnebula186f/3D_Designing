Statement of Originality

I hereby confirm that the 3D Web Application submitted for the Web 3D module Assignment 2026 is my own original work, produced for the purpose of academic assessment.

What is original

  • The complete HTML / CSS / JavaScript layout, navigation, and responsive grid system was written by me.
  • The three 3D bottle models (Azure Lime, Lost Cherry, F* Fabulous) were modelled procedurally in code using Three.js primitives (RoundedBoxGeometry, CylinderGeometry, PlaneGeometry). No bottle meshes were imported from any external source.
  • All textures (perfume labels, marble podium veining) are generated at runtime onto an HTML <canvas> by code I wrote.
  • The PMREM environment map is constructed in-app from a custom GLSL shader pair (vertex + fragment) written for this submission.
  • The colour theming, camera-view animation, bottle gallery carousel, wireframe / spin / headlight toggles, post-processing pipeline, and disposal logic for switching bottles were all written by me.

Brand & visual references

The visual style of the page (dark gradient background, marble podium presentation, serif logotype) is inspired by — but does not copy assets from — the marketing imagery of the Tom Ford Private Blend fragrance line. The product names "Azure Lime", "Lost Cherry" and "F* Fabulous" are existing Tom Ford fragrances and are used here for educational, non-commercial illustrative purposes only. No artwork, photograph, video, audio, font file or 3D mesh has been copied from Tom Ford or any third party into this submission. The Tom Ford brand was deliberately chosen because it is not the brand designed in the labs (Coca-Cola), as required by the assignment brief.
